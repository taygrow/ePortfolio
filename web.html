HTML, JavaScript, and CSS Was Used to Create These Quizzes

Please enjoy my Pop Quiz on Westward Expansion! To create tis quiz, I began by downloading the Javascript file as well as the html file. I followed the suggestions of Dr. L and added in a few breaks in the page to space it out well. I then changed the background color to a brown to match the Westward Expansion theme. I duplicated the code for the first five questions and then went in and edited all ten of the questions to be filled with my Westward Expansion questions. Lastly, I uploaded both the HTML and JS file to a repository that I created in GitHub and followed a tutorial to upload the files and have my website go live.

Westward Expasion Quiz

To work through this assignment, I decided to use Parts of Speech, with the categories Noun, Verb, or Adjective. Therefore, I edited each box to allow for three nouns, three verbs, and three adjectives to fit the categories. I ensured that each word corresponded to the correct box"x" within the code. Next I chose to customize the colors. I changed the background to a darker color and decided to change the colors within the text boxes as well. Finally, I chose to change the correct and incorrect answer colors. I chose to allow the user to have multiple attempts, so I chose to keep the answer selections unlocked. Once I was happy with the colors, I chose to change the font to a more inviting one. I continued to work within the CSS style tab to change the font to Times New Roman- Georgia. I then navigated into a new GitHub repository that I named Drag-n-Drop. I uploaded all three files and added in a ReadMe file. I then ensured that my repository was public. Upon ensuring that my repository was public and that all three files were uploaded, I published my site utilizing the main branch. Then I was able to grab the link to my repository.
